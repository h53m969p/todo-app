# todo-app

## 概要
***
タスクを追加、削除、編集、完了といった操作を行えるシンプルなToDoアプリ

## デモ
***

## 基本機能
***
1. ユーザー登録・ログイン
  アカウント作成機能
  ログイン機能
2. タスクの作成・編集・削除
  新しいタスクの作成
  既存タスクの編集
  タスクの削除
3. タスクの一覧表示
  ユーザーのタスクを一覧表示
4. タスクの完了/未完了の管理
  タスクの完了/未完了のマーク
5. タスクのカテゴリー・優先度分け
   タスクをカテゴリー・優先度別に分類
  カテゴリー・優先度ごとにフィルタリングしてタスクを表示
6. 期限設定
  タスクに締切日を設定
7. 検索機能
  タスク名で検索
